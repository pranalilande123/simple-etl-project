# Simple ETL Project

This project demonstrates a basic ETL (Extract, Transform, Load) pipeline using Python.

## What this project does
- Extracts sales data from a CSV file
- Filters Laptop and Mobile products
- Calculates total sales amount
- Cleans and renames columns
- Loads cleaned data into a new CSV file

## Technologies Used
- Python
- Pandas
- CSV files
- GitHub

## Output
- sales_cleaned.csv
